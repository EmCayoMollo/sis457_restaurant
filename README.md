# sis457_restaurant

El Restaurante ofrece servicios de alimentación, y dispone de un menú variado con diferentes platos
que los clientes pueden consultar de manera fácil y rápida. Las reservas de mesas incluyen la fecha, 
la hora y el número de comensales. Cada plato cuenta con un archivo de fotos que permite a los clientes
visualizar su presentación antes de hacer un pedido o reserva. Los usuarios del sistema tienen roles 
asignados con permisos específicos, permitiéndoles acceder solo a las funciones necesarias según sus 
responsabilidades diarias. Cada cliente puede realizar pagos por sus consumos, los cuales están vinculados
a sus pedidos o reservas. El control de inventario incluye todos los ingredientes, utensilios y recursos 
utilizados en el restaurante, asignados a diferentes áreas como cocina, barra o almacén. Esto permite
mantener un seguimiento constante del uso y estado de cada artículo.
